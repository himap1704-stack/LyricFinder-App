# LyricFinder-App
Aplikasi web sederhana untuk mencari dan menampilkan lirik lagu berdasarkan judul lagu atau nama artis. 
Kamu adalah AI pembuat aplikasi web yang ahli dalam HTML, CSS, dan JavaScript.

Buatkan sebuah website statis berbasis HTML, CSS, dan JavaScript berdasarkan informasi berikut:

### Nama Aplikasi
LyricFinder

### Tujuan / Fungsi Utama
Mencari dan menampilkan lirik lagu secara cepat berdasarkan judul lagu atau nama penyanyi.

### Target Pengguna
Pengguna umum yang suka mendengarkan musik

### Fitur-Fitur yang Diinginkan
1. Pencarian Lirik:
Pencarian berdasarkan judul lagu atau nama artis.
Auto-suggest saat mengetik untuk mempercepat pencarian.

2. Tampilan Lirik:
Lirik lengkap dengan format yang rapi.
Highlight bagian lagu saat diputar (opsional jika ada integrasi musik).

3. Bookmark / Favorit:
Simpan lirik favorit agar bisa diakses offline.

4. History Pencarian:
Menyimpan riwayat pencarian terakhir.

5. Dark Mode / Light Mode:
UI bisa berubah sesuai preferensi pengguna.

### Desain dan Gaya Tampilan
Simple & Minimalis: Fokus ke teks lirik
Warna: Kombinasi biru & putih (default), dark mode hitam & abu-abu
Tipografi: Font yang mudah dibaca untuk lirik
Layout:
Halaman utama: search bar + daftar rekomendasi lagu
Halaman lirik: lirik lengkap + tombol bookmark

### Interaksi dan Perilaku
Input teks → tombol cari → tampil lirik
Klik ikon favorit → simpan lagu
Swipe untuk kembali ke pencarian
Loading animasi saat menunggu hasil

### Teknologi dan Struktur
HTML

### Tambahan
Ikon aplikasi di home screen
Animasi loading saat mencari lirik

### Instruksi Akhir
Buat seluruh aplikasi dalam satu file HTML agar mudah dijalankan.
Gunakan Bootstrap 5 untuk tampilan modern.
Pastikan tampilannya responsive di perangkat mobile.
